Book Description
-----------------

With "Wizards and Warriors: Massively Multiplayer Online Game Creation" you can create your own MMO game, even if you have no programming or game design knowledge. Utilizing the latest version of the popular MMO program Realm Crafter©, simply follow this step-by-step guide and soon you will be playing a game of your own creation, without ever needing to learn complex programming language. Learn what makes a great MMO game, and then get started on yours!
