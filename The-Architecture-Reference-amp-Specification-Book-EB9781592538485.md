Book Description
-----------------

Most architectural standards references contain thousands of pages of details overwhelmingly more than architects need to know to know on any given day. The Architecture Reference &amp;amp; Specification Book contains vital information that's essential to planning and executing architectural projects of all shapes and sizes, in a format that is small enough to carry anywhere. It distills the data provided in standard architectural volumes and is an easy-to-use reference for the most indispensable and most requested types of architectural information.
