Book Description
-----------------

Prepare yourself to take on new and exciting Java programming challenges with this one-stop resource&nbsp;
Job Ready Java&nbsp;delivers a comprehensive and foundational approach to Java that is immediately applicable to real-world environments. Based on the&nbsp;highly regarded and effective&nbsp;Software Guild Java Bootcamp: Object Oriented Programming course, this book teaches you&nbsp;the&nbsp;basic and advanced Java concepts you will need at any entry-level Java position.&nbsp;
With the “Pulling It Together” sections, you’ll combine and integrate the concepts and lessons taught by the book, while also benefiting from:&nbsp;
A thorough introduction&nbsp;to getting set up with Java, including how to&nbsp;write, compile, and run Java programs with&nbsp;or without&nbsp;a&nbsp;Java IDE&nbsp;Practical discussions of the basics of&nbsp;the&nbsp;Java&nbsp;language, including syntax, program flow, and code organization&nbsp;A walk&nbsp;through the fundamentals of Object-Oriented Programming including Classes, Objects, Interfaces, and Inheritance,&nbsp;and how to leverage OOP&nbsp;in Java&nbsp;to create elegant code.&nbsp;&nbsp;Explorations of intermediate and advanced Java concepts, including&nbsp;Maven&nbsp;,&nbsp;unit testing,&nbsp;Lambdas,&nbsp;Streams, and the Spring Framework&nbsp;&nbsp;Perfect for Java novices seeking to make a career transition,&nbsp;Job Ready Java&nbsp;will also earn a place in the libraries of Java developers wanting to brush up on the fundamentals of their craft with an accessible and up-to-date resource.&nbsp;
