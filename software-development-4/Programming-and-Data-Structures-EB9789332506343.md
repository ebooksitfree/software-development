Book Description
-----------------

 This book has been designed for the course on Programming and Data Structures for the undergraduate students of Anna University, Chennai. It integrates the core concepts of C Programming and Data Structures into a single comprehensive textbook. The contents of the book are student-friendly, example-driven and program-oriented. Based on the belief that expertise is achieved by practice, this comprehensive book is enriched with illustrations and about 400 solved programs. Each concept of C Programming and Data Structure is explained easily and the reader is then taken straight to the applications. A student can follow the fundamental concepts and run the programs as illustrated. It does not assume prior knowledge of C programming.        Show and hide more
